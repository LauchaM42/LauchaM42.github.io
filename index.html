Trabajo 2

Primero, creé una base de datos utilizando el programa OpenOffice. Luego, abrí una tabla donde ingresé la información correspondiente. Cada fila se denomina registro y cada columna, campo. Los campos se completaron con palabras clave como: street, que indica el nombre de la calle; price, que representa el precio mensual en libras; furnished, que señala si está amueblado; y availability, que especifica hasta cuándo estará disponible. Una vez que las tablas estuvieron terminadas, creamos el modelo de diseño, asignamos los valores correspondientes y, de esta manera, logramos construir nuestra base de datos.

Trabajo 3

Primero busqué la imagen de un tablero de ajedrez junto con todas las piezas. Luego pegué las cabezas de mis compañeros y de mi profesor en cada una de las piezas. Después, les asigné un nombre a cada una de las fichas. A continuación, añadí un fondo de pasto y coloqué encima una mesa para simular que el tablero estaba sobre ella. Finalmente, añadí los brazos del gato de Scratch y del señor Cactus, mientras tomaban unos mates, para dar la impresión de que estaban jugando al ajedrez.
